# Knowledge Base: AI-Powered Research Assistant

A premium, high-fidelity research platform for multimodal PDF exploration, categorized subject management, and executive-level data visualization. Built for speed, precision, and executive aesthetic.

## 🚀 Quick Start

### Prerequisites
- Node.js v18+
- Google Cloud API Key (Gemini 1.5 Pro + Imagen 4.0)
- Ngrok (for remote access management)

### Installation
```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env
# [Edit .env with your credentials]

# Run development server
npm run dev
```

## ✨ Key Features
- **Multimodal Intelligence**: Query PDFs using Gemini 1.5 Pro with full citation mapping.
- **Visual Synthesis**: Generate research visualisations via Imagen 4.0 integration.
- **Voice-Native UX**: Hands-free research with STT/TTS and interruption protocols.
- **Oatmeal Design System**: A premium Ivory/Sage-Green aesthetic with glassmorphism.
- **Admin Diagnostics**: Executive dashboard for system health, spend monitoring, and design parity.

## 🏗️ Technology Stack
- **Frontend**: React, Vite, Framer Motion, Vanilla CSS Tokens.
- **Backend**: Node.js, Express, SQLite.
- **AI Models**: Google Gemini 1.5 Pro, Imagen 4.0.
- **Authentication**: Google OAuth 2.0.
